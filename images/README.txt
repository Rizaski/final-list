Voter photos folder
==================
Place voter photos here so they show on the Voters and Pledges pages.

Naming: use the voter's ID card number (National ID) as the filename.
- 12345.jpg
- 12345.jpeg
- 12345.png

The app will try .jpg first, then .jpeg, then .png if not found.
Use lowercase "images" as the folder name (this folder).
